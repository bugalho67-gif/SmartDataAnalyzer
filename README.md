# 📊 Smart Data Analyzer

Analisador automático de dados desenvolvido em Python com Streamlit.

## Recursos

- Upload de CSV
- Upload de Excel
- Upload de JSON
- Dashboard
- Estatísticas
- Gráficos
- Correlação
- Outliers
- Relatórios
- Exportação

## Instalação

```bash
pip install -r requirements.txt
```

## Executar

```bash
streamlit run app.py
```

## Tecnologias

- Python
- Pandas
- Plotly
- Streamlit
- Matplotlib
